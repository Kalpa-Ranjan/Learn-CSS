## 1. Hover
Code for HTML
> 1. Make a text "TextHover", Background color: Black, text color: white, on mouse hover make it Red, on click: no response.
```html
<a href="#" class="btn" target="_blank">TextHover</a>
```
Code for CSS
```css
body{
    background-color: black;
    font-size: 32px;   
}

.btn{
    color: white;
}

.btn:hover{
    color: red;
}
```
